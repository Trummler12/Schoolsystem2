<p>The following <a href="/wiki/Index_(publishing)" title="Index (publishing)">index</a> is provided as an overview of and topical guide to science: Links to articles and redirects to sections of articles which provide information on each topic are listed with a short description of the topic. When there is more than one article with information on a topic, the most relevant is usually listed, and it may be cross-linked to further information from the linked page or section.
</p><p><b><a href="/wiki/Science" title="Science">Science</a></b> (from <a href="/wiki/Latin_language" class="mw-redirect" title="Latin language">Latin</a> <i>scientia</i>, meaning "knowledge") is a systematic enterprise that builds and organizes <a href="/wiki/Knowledge" title="Knowledge">knowledge</a> in the form of testable <a href="/wiki/Explanation" title="Explanation">explanations</a> and <a href="/wiki/Predictions" class="mw-redirect" title="Predictions">predictions</a> about the universe.<sup id="cite_ref-Wilson-1999b_1-0" class="reference"><a href="#cite_note-Wilson-1999b-1"><span class="cite-bracket">[</span>1<span class="cite-bracket">]</span></a></sup><sup id="cite_ref-Heilbron-2003_2-0" class="reference"><a href="#cite_note-Heilbron-2003-2"><span class="cite-bracket">[</span>2<span class="cite-bracket">]</span></a></sup>
</p><p>The <a href="/wiki/Branches_of_science" title="Branches of science">branches of science</a>, also referred to as scientific fields, scientific disciplines, or just sciences, can be arbitrarily divided into three major groups:
</p>
<ul><li>The <a href="/wiki/Natural_science" title="Natural science">natural sciences</a> (<a href="/wiki/Biology" title="Biology">biology</a>, <a href="/wiki/Chemistry" title="Chemistry">chemistry</a>, <a href="/wiki/Physics" title="Physics">physics</a>, <a href="/wiki/Astronomy" title="Astronomy">astronomy</a>, and <a href="/wiki/Earth_science" title="Earth science">Earth sciences</a>), which study <a href="/wiki/Nature" title="Nature">nature</a> in the broadest sense;</li>
<li>The <a href="/wiki/Social_sciences" class="mw-redirect" title="Social sciences">social sciences</a> (e.g. <a href="/wiki/Psychology" title="Psychology">psychology</a>, <a href="/wiki/Sociology" title="Sociology">sociology</a>, <a href="/wiki/Economics" title="Economics">economics</a>, <a href="/wiki/History" title="History">history</a>) which study people and societies; and</li>
<li>The <a href="/wiki/Formal_sciences" class="mw-redirect" title="Formal sciences">formal sciences</a> (e.g. <a href="/wiki/Mathematics" title="Mathematics">mathematics</a>, <a href="/wiki/Formal_logic" class="mw-redirect" title="Formal logic">logic</a>, <a href="/wiki/Theoretical_computer_science" title="Theoretical computer science">theoretical computer science</a>), which study abstract concepts.</li></ul>
<p>Disciplines that use science, such as <a href="/wiki/Engineering" title="Engineering">engineering</a> and <a href="/wiki/Medicine" title="Medicine">medicine</a>, are described as <a href="/wiki/Applied_sciences" class="mw-redirect" title="Applied sciences">applied sciences</a>.
</p>
