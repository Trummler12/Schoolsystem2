# Projektaufgabe M320 & M322

**Projektbeschreibung**: In dieser Projektarbeit entwickelst du deine eigene Webapplikation mit einem Backend in Java und einem Frontend in HTML & JavaScript. Du planst, entwirfst, programmierst und dokumentierst eine vollständige Anwendung. Dabei wendest du die Konzepte aus den Modulen M320 (Objektorientierte Programmierung) und M322 (Benutzerschnittstellen) an.

Für eine einfache Umsetzung empfehlen wir einen Technologiestack, bei dem das Java-Backend eine REST-Schnittstelle bereitstellt (z.B. mit einem schlanken Framework wie Javalin), die von deinem Frontend (HTML, CSS & JS) genutzt wird. Ziel ist es, die Konzepte der objektorientierten Programmierung und des User-Interface-Designs sinnvoll zu verbinden und ein funktionierendes System zu bauen, das du von Grund auf erklären kannst.

**Zeit**: 7 Wochen

**Start**: 27.10.2025

**Abgabe**: 08.12.2025

**Präsentation**: 15.12.2025

**Anforderungen**: Basierend auf den Kompetenzmatrizen M320 und M322.

**Technologie**: Java, HTML, JS
